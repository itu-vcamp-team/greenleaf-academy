# Frontend Yol Haritası (roadmap/frontend/overview.md)

## 1. Aşama: Marka Kimliği ve Auth Akışı
- [ ] **Tema Yapılandırması**: Tailwind 4 ve tenant duyarlı dinamik renk paleti.
- [ ] **Landing Page**: POC tarzı, etkileşimli Misafir (Guest) sayfası.
- [ ] **Auth UI**: Kayıt (Referans kodu + Partner ID) ve Giriş ekranları.
- [ ] **Middleware**: Subdomain tabanlı tenant tespiti ve yönlendirme mantığı.

## 2. Aşama: Akademi ve Takvim Deneyimi
- [ ] **Reels Player**: YouTube IFrame API entegrasyonu, sessiz otomatik oynatma ve ilerleme barı.
- [ ] **Academy Dashboard**: Kategori bazlı içerik listeleme ve kilitli içerik görselleştirmesi.
- [ ] **Smart Calendar**: Etkinlik listesi ve "Takvime Ekle" butonları.
- [ ] **Document Viewer**: Google Drive embed döküman görüntüleyici.

## 3. Aşama: Partner ve Admin Panelleri
- [ ] **Partner Dashboard**: Aday ilerleme takibi ve kişisel profil yönetimi.
- [ ] **Admin Panel**: İçerik yükleme (YouTube link + PDF link), partner onaylama, etkinlik oluşturma.
- [ ] **Super Admin UI**: Manuel tenant kurulum rehberi ve temel yapılandırma ekranları.
- [ ] **Responsive Design**: Tüm ekranların "Mobil Öncelikli" saha kullanımına optimizasyonu.
